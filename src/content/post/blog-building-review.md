### 前言

　　很早就有搭建个人站的打算，不过碍于没有相关专业知识，加上自认为不是写文章的料，更多的是为了一点新鲜感，总之很多次半途而废。不过也是因为想做一些分享，单纯记录也好，于是就开干，一步步实现出来也颇有趣味。

### 实现历程

　　首先就是可以有自己的域名喽，openora 是让 ChatGPT 帮取的，我想既然是分享，那配上开放而又闪光的 tag 不过分吧，中文名也是想了会，就称之为思维绽放吧。搭配 org 顶级域名，名起的有点大了，~~这么一看挺唬人的~~。

　　域名是在 Cloudflare 上买的，搭配其 DNS 解析等功能，一站式解决各种问题，总之便利就对了~

　　服务器一开始买了大厂的 VPS，准备自己搭建，鼓捣了一番，但不是专职运维，自行维护总归是一个额外负担，加之对我来说自身 VPS 利用率不高，还是找找其他便宜又便利的方式吧。最终使用了 Github Pages，结合 Git 仓库推送，自动部署。当然只能是静态站，不过这对于我来说够用了，调整好模板，只要专注于文章就行了。

　　同时要挑选博客模板，带着静态站的关键词几经搜索发现了 Astro，找找模板，诶 Cactus theme 看起来很对眼，简洁风。

　　模板定下来了，那么网站可能会有一些大的资源文件，不可能都上传 Github，那就需要将大文件存储到专门的储存库，找到 Cloudflare 家产品 R2 存储，免费 10G，大文件就放这了。

　　接下来就是繁琐的模板调试了，面对眼花缭乱的前端代码，看起来还是很费劲的，好在有 AI 的帮助，解决了很多问题。

　　到这里搭建就告一段落了，接下来专注内容吧。
